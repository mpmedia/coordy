Node3d()constructor
public function Node3d(link:Object = null, x:Number = 0, y:Number = 0, z:Number = 0, jitterX:Number = 0, jitterY:Number = 0, jitterZ:Number = 0)

Base node for all 3D layouts. Contains basic information for recording and projecting 3D positioning. Used in Sphere3d and Snapshot3d layouts.

Parameters
link:Object (default = null) — The node's target object
 
x:Number (default = 0) — Node's x position
 
y:Number (default = 0) — Node's y position
 
z:Number (default = 0) — Node's z position
 
jitterX:Number (default = 0) — Node's x-jitter value
 
jitterY:Number (default = 0) — Node's y-jitter value
 
jitterZ:Number (default = 0) — Node's z-jitter value
